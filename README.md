## Installation
```shell
git clone https://github.com/himiko3939/net.git
```
```shell
cd Netcoin-BOT
```
```
npm install
```

## Usage

### Edit data.txt
```shell
nano data.txt
```
### Create screen
```
screen -S your_screen
```
### Run
```
node main.js
```
